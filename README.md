## Project Description
This project is a Tenant Management Web Application built using Node.js and Express.
It allows landlords to manage tenants, rooms, and rent payments.

## Features
- Add tenant
- View tenants
- Delete tenant
- Rent displayed in TZS
- Desktop application support

## How to Run
1. Open terminal in project folder
2. Run:
   node server.js
3. Open browser:
   http://localhost:5000
